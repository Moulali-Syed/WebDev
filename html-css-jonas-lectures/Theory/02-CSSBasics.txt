3 ways of styling:
=================
Inline style
	<h1 style="color:blue">Hello</h1>

	never preferable

Internal CSS
	in head tag write style tag
	<style>
		h1{
			color:blue;
		}
	</style>

External CSS
	create style.css file and link it to html file
	and write css in the style.css

	 <link rel="stylesheet" href="style.css" />

to select first paragraph inside a article element

  font-style: italic;
}
p must be direct child of article , else it will not work

to select last-child of article which is paragraph

  color: red;

Styling links:
==============
/* targets only those anchor elements that has href attribute */
a:link {
  color: #1098ad;
}

remove underline of link
 text-decoration: none;

 /* targets only those anchor elements that has href attribute */
a:link {
  color: #1098ad;
  text-decoration: none;
}

a:visited {
  color: #1098ad;
}

a:hover {
  color: orangered;
  font-weight: bold;
  /* text-decoration: underline dotted orange; */
  text-decoration: underline orangered;

/* when we click on link */

Conflicts between Selectors:
=============================

if multiple last selector in code applies 

Inheritance and universal selector:
==================================
some properties get inherited from parent to child elements
body properties will get inherited to child
But not every property is inherited and also can be overwritten 
Not all properties get inherited , mostly ones related to text font-family, font-size , font-weight
font-style , line-height , letter-spacing , text-align , text-transform , text-shadow , list-style,..etc

border property doesnot get inherited

universal selector selects every element
*{

}

universal selector properties doesnot get inherited , but properties of body get inherited
